# Instrucciones del Proyecto

## Regla Principal: Frontend Design First

**OBLIGATORIO**: Antes de realizar cualquier tarea relacionada con frontend (componentes, páginas, estilos, UI/UX), DEBES:

1. Invocar el skill `/frontend-design` usando la herramienta Skill
2. Leer y seguir completamente el workflow que proporciona el skill
3. No comenzar a escribir código hasta completar el proceso de diseño del skill

### Tareas que requieren /frontend-design:
- Crear componentes React/HTML/CSS
- Modificar estilos o layouts
- Construir páginas o secciones
- Diseñar interfaces de usuario
- Landing pages, dashboards, formularios
- Cualquier trabajo visual o de UI

### Workflow esperado:
1. Usuario solicita tarea de frontend
2. Claude invoca `/frontend-design`
3. Claude sigue las instrucciones del skill
4. Solo entonces Claude implementa el código
